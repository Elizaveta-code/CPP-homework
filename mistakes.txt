1) Отсутствует #include <stdio.h>. Функция printf взята из библиотеки stdio.h, поэтому ее нужно подключить с помощью директивы
2) Пропущен амперсанд в scanf("%i", &a)
3) Надо 2 знака равно в условии if (a == 10)
4)Надо убрать символ переноса строки (\n) в форматной строке, надо scanf("%i", &a)
Символ \n в форматной строке scanf означает:
а) Считать число, затем
б) Пропустить все пробелы, табуляции, переносы строк до первого непробельного символа
в) Программа будет ожидать ввода, пока не получит непробельный символ после числа, что создает впечатление запроса второго числа
5)Без фигурных скобок {} оператор if применяется только к следующему за ним оператору. Второй printf выполняется всегда, независимо от условия
6)Точка с запятой после условия if. 
if (a > 0); - это условие с пустым телом, printf("Positive\n"); - выполняется всегда, независимо от условия
7) i никогда не увеличивается, тк нет счетчика i++
8) Точка с запятой после условия цикла while
9) Неправильный, для 5 ответ = 1884844111. Переменная sum без инициализации содержит "мусорное" значение из памяти
Результат зависит от того, что было в этой ячейке памяти ранее. Надо sum = 0;
10) Выход за границы массива в цикле. Правильно будет for (int i = 0; i < 5; ++i)
