После ввода числа и нажатия Enter в буфере ввода остается символ новой строки \n. Когда выполняется второй scanf("%[^\n]", str), он сразу встречает \n и завершает чтение, не считывая нужную строку.
Можно добавить пробел
#include <stdio.h>
int main()
{
    int a;
    scanf("%i", &a);
    printf("a = %i\n", a);

    char str[100];
    scanf(" %[^\n]", str);  // пробел перед % поглощает пробельные символы
    printf("str = %s\n", str);
}
